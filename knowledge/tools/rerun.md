# Rerun - Screen Capture & Search Tool

## Overview
Rerun is a macOS application that quietly captures screen activity and makes it all searchable via AI-powered OCR. Created by Josh Pigford, it allows users to find anything they've seen on their screen without losing privacy.

## Key Features
- **Screen Capture:** Automatically captures screen content in the background
- **AI Search:** Uses AI to make captured content searchable
- **OCR:** Optical Character Recognition to index text from visual content
- **Privacy-First:** All data remains on your local Mac
- **Apple Silicon Optimized:** Built for modern Mac hardware

## Status
Early release/beta version (as of January 2026) - seeking user feedback and testing

## Use Cases
- Finding information you've seen on screen previously
- Searching screenshots and visual content
- Privacy-respecting screen recall without cloud storage
- Reference material lookup
