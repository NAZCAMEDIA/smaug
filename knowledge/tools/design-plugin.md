Claude Code plugin for design and refinement workflows. Provides a streamlined process for designing interfaces and iterating on designs within the Claude Code environment.

## Key Features

- `/design-and-refine:start` command initiates the design workflow
- Automated design process integration
- Simplifies UI/UX design within terminal-based development
- 100 GitHub stars

## Usage

Install the plugin and use the slash command:
```bash
/design-and-refine:start
```

The plugin handles the rest of the design refinement process.
