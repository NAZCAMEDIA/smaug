# QMD - Query Markdown

**Repository:** https://github.com/tobi/qmd

**Creator:** Tobi

## Overview

Tool that enhances Claude Code's ability to understand and connect information in Obsidian vaults beyond basic text search capabilities (grep/glob).

## Key Capabilities

- Enhanced information retrieval from markdown notes
- Better connection discovery between ideas
- Integration with Obsidian + Claude Code workflow
- Improved contextual understanding of note relationships

## Use Cases

- Obsidian knowledge base querying
- Enhanced context understanding for Claude Code
- Connecting disparate notes and ideas
- Building comprehensive view of knowledge relationships

## Integration

Works seamlessly with:
- Obsidian vault setup
- Claude Code CLI
- Markdown note systems

## Context

Shared by Eric Hu as part of an Obsidian + Claude Code workflow optimization. Meaningfully improves Claude's ability to surface connections and patterns in knowledge bases.
