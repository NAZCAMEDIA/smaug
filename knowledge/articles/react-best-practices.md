A comprehensive React best practices guide released by Vercel as a reusable Skill for Claude Code. This resource, colloquially referred to as the "React bible," provides development guidelines, patterns, and recommendations for building high-quality React applications.

## Overview

The React Best Practices Skill is part of Vercel's effort to provide developers with structured, expert-validated guidance for React development. It's designed to be used directly within Claude Code, providing real-time recommendations and patterns while coding.

## Installation

The Skill can be installed with a single command in Claude Code:

```bash
npx claude-code-templates@latest --skill=web-development/react-best-practices --yes
```
